## Usage

Simply open `index.html` in your browser. The tool provides three main functions:

1. **Paste**: Click the "Paste" button or press Ctrl+V (Cmd+V on Mac) to load HTML from your clipboard
2. **Edit**: Type or modify the HTML code directly in the editor
3. **Copy**: Click the "Copy" button to copy the current HTML back to your clipboard

## Security Note

This tool is a code editor and does not render or execute HTML content. It safely displays HTML as editable text.
